The process for generating the dashboard could be found as following: 
1. Read into samples.json data;
2. Use d3.json to fetch sample data for the plots;
3. Build Bubble Chart by using the sample data;
4. Build the bar Chart;
5. Use the list of sample names to populate the select option;
6. Use the first sample on the list to build the initial plots;
7.  Fetch new data when the new sample is selected.
